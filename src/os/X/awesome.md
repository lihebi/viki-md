# Awesome Window Manager

# Shortcuts

Control

* `Mod-C-r`: restart
* `Mod-S-q`: quit
* `Mod-r`: run prompt
* `Mod-return`: open terminal emulator

Client

* `Mod-m`: maximize client
* `Mod-n`: minimize client
* `Mod-f`: full screen

Navigation

* `Mod-j`: Focus next client
* `Mod-k`: Focus previous client
